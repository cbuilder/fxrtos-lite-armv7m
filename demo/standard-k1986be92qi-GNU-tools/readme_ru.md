Демонстрационное приложение мигание светодиодами из потоков для сборки инструментами GNU
========================================================================================

Описание
--------

Демонстрационное приложение мигание тремя светодиодами из разных потоков. Диоды переключаются с различными частотами.

Список файлов
-------------

| File name         | Description
|-------------------|---------------------------------------------------------
| build.bat         | сценарий сборки
| FXRTOS.h          | заголовочный файл интерфейса FX-RTOS
| fxrtos_demo.elf   | собранное приложение (ELF формат)
| fxrtos_demo.hex   | образ приложения для flash памяти устройства
| libfxrtos.a       | ядро FX-RTOS
| main.c            | приложение
| startup.S         | таблица векторов и начальная инициализация
| core_cm3.h        | описание ядра ARM Cortex M3
| system_MDR32F9Q.h | интерфейс функций настройки частоты
| system_MDR32F9Q.c | реализация функций настройки частоты
| MDR32F9Qx_config.h| определения для настройки частоты
| MDR32F9Qx_lib.h   | определения библиотеки периферии микроконтроллера

Сборка
------

Предварительные условия: компилятор и компоновщик должны быть доступны через переменную PATH.

1. Установить переменную окружения GCC_PREFIX для используемого компилятора. Пример: set GCC_PREFIX=arm-none-eabi-
2. Выполнить build.bat

Результат: должен появиться загружаемый файл HEX в папке приложения.

Настройка оборудования
----------------------
Плата LDM-K1986BE192QI может быть запитана от USB порта ПК.
Выберите подачу питания от USB-A с помощью джампера на разъеме XS1 (USB-A).
SW[2:0] переключатель должен стоять в положении соответсвующем двоичному значению 101.
Этот способ передает прошивку через порт UART2 микроконтроллера. Этот порт подключен к микросхеме виртуального COM-порта на плате.
Включить подачу питания кнопкой SW11 (ON/OFF). 

Загрузка в устройство
---------------------
Собранный файл прошивки может быть загружен во Flash память с помошью утилиты 1986WSD.exe (Windows).
Пользователь должен указать путь к файлу прошивки, используемый COM порт и нажать кнопку Start или Start+Run.
